Sentimental Analysis

This application is used to analyse an english text per paragraphs and generate a chart of the sentiments of each paragraph of the text. Any kind of text can be used but it fits better to texts that contains opinion about something.

When the paragraph is identified as positive it gets a number 1, otherwise it gets a 0. Currently on beta version we are just using the first 10 paragraphs of the text, so if your text is bigger only the first 10 ones will be used.
